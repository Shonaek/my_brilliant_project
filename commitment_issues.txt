How to commit things:

Committing for the first time:

1. in the git pane in the top right, click on the 'stage' button next to the file
2. click on the 'diff' button to open up the commit screen
3. write a commit message
4. click commit, then close the pop-up window


Subsequent commits:

1. edit the file
2. save the file
3. click the 'stage' button in the top right pane
4. click on 'diff'
5. enter a commit message
6. click commit, and close the pop-up window
